Dorian Yates' Blood & Guts 6-Week Trainer

Created by legendary pro bodybuilder and six-time Mr. Olympia Dorian Yates, Blood & Guts is a classic, aggressive bodybuilding program for maximizing muscle growth. This six-week plan is built on Dorian's signature training style: high-intensity training, or HIT. And to demonstrate, he enlists renowned trainer and bodybuilder Kris Gethin as the victim.

Program Overview

Learn high-intensity training from the master.

4 workouts per week / 45-60 min. workouts

Training Split Example

  • Day 1: Chest and Biceps
  • Day 2: Back
  • Day 3: Rest Day (Cardio)
  • Day 4: Shoulders and Triceps
  • Day 5: Legs
  • Day 6: Rest day (Cardio)
  • Day 7: Rest Day (Cardio)

Workout Example

  • Tredmill waking - 10 mins
  • Rotator cuff warm-up - 2 sets, 12 reps (rest 1 min.)
  • Decline barbell bench press: 2 sets - 10-12 reps, 1 set - 6-8 resps to failure (rest 1min.)
  • Incline dumbell bench press: 1 set - 10-12 reps, 1 set - 6-8 reps to failure (rest 1min.)
  • Dumbell chest fly: 1 set - 10-12 reps, 1 set - 6-8 reps to failure (rest 1 min.)
  • Concentration curl: 1 set - 10-12reps per each side, 1 set - 6-8 reps to failure per each side (no rest)
  • Barbell Curl: 1 set - 6-8 reps to failure
